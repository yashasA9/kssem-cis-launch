# IEEE CIS Inaugural Expo

Interactive launch experience for the IEEE CIS Chapter.

The interface visualizes computational intelligence by constructing
a neural brain structure as users answer quiz questions correctly.

## Tech Stack

React  
Three.js  
React Three Fiber  
GSAP  
TailwindCSS

## Features

• Interactive launch experience  
• Particle-based neural brain formation  
• Cinematic animations  
• Voice feedback on completion

## Run locally

npm install
npm run dev
